# Artificial Neural Network Classification

## Overview

This project implements an Artificial Neural Network (ANN) for solving a supervised classification problem using TensorFlow and Keras. The project demonstrates data preprocessing, neural network architecture design, model training, and evaluation.

---

## Objective

Build a deep learning model capable of learning complex patterns within structured data and producing accurate predictions.
